Input file
	
	Either pre-aligned raw text sequences, FASTA peptides,
		pre-algined FASTA sequences, or Maxquant evidence.txt
	Center: boolean

Context

	FASTA.

Background

	List of sequences, or list of background frequencies
		supplied as CSV or JSON.
	Optional: Either explicit None argument or a dictionary
		of compound residues to override default compound
		residues.

PatternContainer

	sequence_tensor,
	background,
	initial_pattern=None,
	max_depth=None,
	p_value_cutoff=0.01,
	occurrences=2,
	fold_change_cutoff=1,
	multiple_testing_correction=True,
	positional_weighting=True


INPUT ARGUMENT LIST

x title (string)
x decription (string, optional)
x email (string, optional)
x data (file)
x select data format (pre-aligned raw text, fasta peptides, prealigned
	fasta sequences, maxquant evidence.txt)
x context (file, optional)
x background (file, optional)
x select background format (fasta, frequency list)
x background regular expression (string, optional)
x expand peptides (boolean)
x center positions (boolean, optional)
x compound residues (boolean, optional)
x max depth (int, optional)
x p-value cutoff (float, optional)
x minimum occurrences (int, optional)
x fold change cutoff (float, optional)
x multiple testing correction (boolean, optional)
x positional weighting (boolean, optional)
